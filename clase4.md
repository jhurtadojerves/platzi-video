#¿Porqué aprender React?

- Es una librería de Javascript que nos ayuda a construir interfaces de usuario.

- ¿Porqué? Las necesidades del frontend han aumentado durante el paso de los años. El enfoque del desarrollo web se ha orientado a la experiencia de usuario. Se usa para interfaces muy dinámicas.

###Backbone.js

- Apoyaba a jquery (que permitia enlazar eventos) permitiendo manejar los datos de la interface.

###Angular, ember, meteor.
Nacen framworks para unificar las funcionalidad de jquery y backbone sin ser dependientes de ellos. 

##Angular 2.0, React, Vue.js

- Angular 2 cambió totalmente a nivel de API.

- Tienen un enfoque orientado a componentes, permite que la aplicación sea más matenible.

##React, características
- Es declarativo
- Está basado en componentes. 
- Aprende una vez, escribe dónde quieras. No hace falta mudar toda la aplicación a React.

React se encarga únicamente de construír interfaces.
